常见问题

原因：旧版本未适配新系统权限机制

解决方案：

  • 下载2.4蓝色版
  • 或使用派大星3.0适配版
  • 临时方案：设置 → 开发者选项 → 开启「停用HW叠加层」

解决方法：

  1. 切换至系统默认输入法
  2. 禁用剪贴板保护功能
  3. 检查规则首尾是否为[{...}]结构
  4. 使用在线文档同步

预防措施：

  • 每次更新前通过「规则管理」→「导出配置」保存本地
  • 建议使用云备份功能
备份路径：
/Android/data/ltiapp/config

解决方案：

  1. MIUI系统：设置 → 电池优化 → 选择「无限制」
  2. 定期检查通知栏服务图标状态
  3. 关闭手机管家等安全软件的自动优化功能

同步方案：

方案一：云同步
  • 需注册开发者账号
  • 支持实时同步规则配置
方案二：手动同步
  • 复制文件夹：/Android/data/ltiapp/config
  • 建议使用MT管理器操作

版本 特点 适用系统
2.2派大星 稳定版（停更维护） 安卓7-11
2.4蓝色版 支持安卓12/13 需ROOT权限
3.0测试版 AI学习广告（测试功能） 安卓14+

配置步骤：

  1. 规则管理 → 添加 com.tencent.wework
  2. 启用「深度跳过」选项
  3. 关闭企业微信的「安全键盘」
注意：需要企业微信版本 ≥ 4.1.2
